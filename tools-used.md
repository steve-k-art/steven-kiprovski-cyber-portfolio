# Cybersecurity Tools Used

## Kali Linux

Primary operating system used for penetration testing and vulnerability assessment.

## Nmap

Used for network discovery and service enumeration.

## Burp Suite

Used to intercept and manipulate HTTP requests during web application security testing.

## Nikto

Used to scan web servers for vulnerabilities and outdated configurations.

## Metasploit Framework

Used to test exploitation of identified vulnerabilities.

## Wireshark

Used for analysing network traffic and packet captures.
